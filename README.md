# SQL Básico – Práctica

Este repositorio contiene ejercicios básicos de SQL enfocados en
consultas comunes usadas en entornos reales (reportes y análisis).

## Motor
- MySQL 

## Tablas usadas
- actores(id, nombre)
- series(id, titulo)
- episodios(id, serie_id, rating_imdb, duracion)
- actuaciones(actor_id, serie_id)

## Consultas incluidas
- Actor/actriz con más series
- Serie con mejor rating promedio
- Episodio con mayor duración

## Objetivo
Demostrar el uso de:
- SELECT
- GROUP BY
- ORDER BY
- COUNT
- AVG
- DISTINCT
